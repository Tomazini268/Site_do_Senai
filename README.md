Desafio SENAI - Página Responsiva

Este projeto foi desenvolvido como parte do desafio proposto pelo curso do SENAI-SP, com o objetivo de aplicar os conhecimentos adquiridos em HTML, CSS e responsividade.

Prévia do Projeto

A proposta era reproduzir a página conforme o modelo apresentado na imagem de referência, garantindo que ela funcionasse bem em diferentes tamanhos de tela (layout responsivo).

Tecnologias Utilizadas

HTML5

CSS3

Flexbox

Media Queries

Design Responsivo

Estrutura da Página

A página foi dividida nas seguintes seções principais:

Cabeçalho (Header)

Logotipo do SENAI

Navegação com links: Home, Cursos, Contato e botão de "Cadastre-se"

Banner Principal

Imagem de fundo com efeito escurecido

Card sobreposto com:

Título: Aprendizagem Online

Botão de ação: Acessar

Conteúdo Informativo

Texto explicativo sobre programação

Imagem ilustrativa ao lado do texto (com efeito de sombra)

Rodapé (Footer)

Logotipo do SENAI

Informações de contato

Direitos autorais

 Ícone flutuante do WhatsApp

Posicionado no canto inferior direito da tela

Responsivo e fixado para facilitar o contato

Responsividade

Foram utilizadas media queries para adaptar o layout a diferentes dispositivos, como:

Smartphones

Tablets

Notebooks e desktops

Os elementos são reorganizados em telas menores para manter a legibilidade e usabilidade.
